Detalji o kursu i rasporedu predavanja se nalaze na sledecem linku:

https://vojvodinaictcluster.org/sr/netjuniorprogram/

Kurs je podeljen na dva modula, a u okviru svakog modula se obradjuje nekoliko podtema: 

1) Osnove programiranja
   1.1) Osnove proceduralnog programiranja
   1.2) Osnove obejtno-orijentisanog programiranja
   1.3) Osnove baza podataka

2) Napredno programiranje
   2.1) Web Frontend Development
   2.2) Principi razvoja softvera i Web Backend Development

To je ono sto je napisano na sajtu, ali ce, izgleda, sam kurs biti nesto drugacije organizovan. Vise o tome znacemo vremenom.
